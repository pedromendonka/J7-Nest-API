# This Repository

## Description

### A boilerplate based in NestJS using Fastify and pre-config with:
- Security
  - Helmet
  - CORS
- Nest ConfigModule for environment variables using validation with Joi
- Tasks using Nest Schedule for
  - CRON
  - Interval
  - Timeout
- Prisma for Database
- Documentation:
  - API docs using Compodoc 
  - REST docs using Swagger

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Build

```bash
# building for production
$ npm run build
```

This `script` will automatically run two other `scripts`:
- `lint` that will do all Eslint checks
- `format` that will format the code with Prettier

After this, it will run `prebuild` `script` that cleans the `dist` folder before building itself.

## Documentation

### CompoDoc
To access CompoDoc documentation just run Compodoc server in parallel with your App.
Just run pre-configure `compodoc` node script:

```zsh
# Documentation
$ npm run compodoc
```
This will generate a folder for all the documentation in the root of your app. To access the full documentation, open your browser and head to your localhost at default port `8080`

### Swagger
Swagger will run automatically in development at `/documentation` or other endpoint you define in SwaggerModule setup in `main.ts`.

## Prisma Init
```zsh
$ npx prisma init
```
